GitHubSearch
===
This is a [React.js](https://reactjs.org/) project for browsing [Github](https://github.com/) user repository page using github API.
## Project Structure:
```
.
src
├── Components  元件庫
│   ├── Home   主頁面
│   ├── Results  repository列表
│   └── RepoDetails    repository頁面
├── App頁面控制
└── index 頁面渲染

```
